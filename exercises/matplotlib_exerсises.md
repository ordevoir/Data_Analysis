### 1. Линейный график функции

Постройте график функции
$$
y = x^2
$$
на отрезке $[-5, 5]$ с шагом $0.1$.

**Требования:**
* Подписать оси к осям (axis)
* Добавить заголовок (title)
* Включить сетку (grid)
* Сделать линию красной и пунктирной

### 2. Несколько графиков на одной диаграмме

На одном диаграмме изобразите графики функций на отрезке $[0, 2\pi]$:

* $y_1 = \sin(x)$
* $y_2 = \cos(x)$
* $y_3 = \sin(x) + \cos(x)$

**Требования:**
* Для каждой функции – свой цвет и стиль линии
* Добавить легенду
* Подписать оси и заголовок
* Включить сетку с мелкой разметкой (`which='both'`)

### 3. Точечный (scatter) график с цветами по категориям

Сгенерируйте 100 случайных двумерных точек $(x, y)$ из нормального распределения.  Каждой точке случайно присвоить один из трёх классов: `A`, `B`, `C`

* Построить диаграмму с разбросом точек (scatter):
  * класс `A` – один цвет и маркер
  * класс `B` – другой цвет и маркер
  * класс `C` – третий
* Добавить легенду с названиями классов
* Подписать оси, заголовок
* Сделать точки полупрозрачными (`alpha`)

### 4. Столбчатая диаграмма

Есть данные о количестве продаж по дням недели:

* Пн: 15
* Вт: 22
* Ср: 13
* Чт: 27
* Пт: 30
* Сб: 45
* Вс: 38

Построить диаграмму с двумя графиками.
1. На первом графике (subplot) – обычная столбчатая диаграмма (вертикальная).
2. На втором подграфике – горизонтальная (`barh`) с теми же данными.
3. Подписать ось категорий (дни недели).
4. Увеличить размер фигуры так, чтобы подписи не налезали друг на друга.
5. Добавить заголовок ко всей фигуре (например, `Weekly Sales`).

### 5. Гистограммы двух выборок

Сгенерируйте две выборки:

* `A` – 1000 значений из нормального распределения $\mathcal{N}(0, 1)$
* `B` – 1000 значений из нормального распределения $\mathcal{N}(2, 1.5)$

* Нарисовать на одной диаграмме две полупрозрачные гистограммы с одинаковыми границами бинов (bins). 
* Добавить легенду (`Sample A`, `Sample B`)
* Добавить вертикальные линии в местах средних значений выборок
* Подписать оси
